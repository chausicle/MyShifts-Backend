# MyShifts-Backend
A backend API for an app that employees can use to get their shifts covered or find shifts they can cover.

Tecnologies used:
- Express.js (Server)
- PostgreSQL (Database)
- bcrypt.js (Authenication)
- JWT (Authorization)
- Sendgrid (Sending email notification to users)

To run it locally:
1. Fork and clone
2. npm install
3. npm start
